# Stepik-Project-Notepad

Учебный проект для управления заметками. Поддерживает создание, редактирование, удаление и просмотр
заметок с возможностью добавления изображений из галереи, полнотекстового поиска и закрепления
важных заметок. Данные хранятся в локальной БД (Room).

## Стек технологий

- **Jetpack Compose** — декларативный UI
- **Room** — локальная база данных
- **Hilt** — внедрение зависимостей
- **Coroutines + Flow** — асинхронность и реактивность
- **Navigation Compose** — навигация между экранами
- **Coil** — загрузка изображений
- **Material 3** — дизайн-система

Архитектура: **MVVM + Clean Architecture** с разделением на feature-модули.

## Требования

- **Android Studio** Meerkat (2025.1) или новее
- **JDK 17+**
- **Android SDK 36** (установится автоматически через SDK Manager)
- Устройство или эмулятор с **Android 7.0 (API 24)** или выше

## Сборка и запуск

1. Склонировать репозиторий и открыть корневую папку проекта в Android Studio.
2. Дождаться завершения Gradle Sync.
3. Выбрать конфигурацию **app** и нажать **Run**.

Сборка release-APK через терминал:

```bash
./gradlew assembleRelease
```

Собранный APK появится в `app/build/outputs/apk/release/`.

## Структура проекта

```
├── app/                     # Точка входа приложения (MainActivity, NavGraph)
├── core/
│   ├── domain/              # Бизнес-логика: модели (Note, ContentItem), use cases, интерфейс репозитория
│   ├── data/                # Room БД, DAO, реализация репозитория, ImageFileManager, Hilt-модули
│   └── ui/                  # Тема (Material 3), цвета, общие UI-компоненты
├── feature/
│   ├── notes/               # Экран списка заметок (поиск, закреплённые, карточки)
│   ├── creation/            # Экран создания заметки (текст + изображения)
│   └── editing/             # Экран редактирования заметки
└── gradle/libs.versions.toml  # Централизованное управление версиями зависимостей
```
